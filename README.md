# Treasure Island Game

Welcome to the Treasure Island game! This is a simple text-based adventure game where you embark on a quest to find hidden treasure.

## Description

In this game, you will navigate through different scenarios by making choices that will determine your fate. Your decisions will lead you to various outcomes, including finding the treasure or facing unfortunate events.

## How to Play

1. Run the game by executing the `treasure_island.py` file.
2. Follow the prompts and make your choices by typing the indicated options.
3. Explore the different paths and see if you can find the treasure!

## Files

- `treasure_island.py`: Contains the main code for the game, including game logic and user interactions.

## Installation

To play the game, ensure you have Python installed on your machine. Clone this repository and run the game using the following command:

```bash
python treasure_island.py
```
